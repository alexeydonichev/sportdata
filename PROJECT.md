# YourFit Analytics Platform

## Суть
Внутренняя аналитическая платформа для бренда YourFit (фитнес-инвентарь).
Замена Google Таблиц, 1С, Битрикс24 — всё в одном месте.

## Маркетплейсы
- Wildberries
- Ozon  
- Яндекс Маркет
- Авито

## Источники данных
- API маркетплейсов (автоматическая сборка)
- Google Таблицы (начальный импорт + ручные данные)
- 1С (себестоимость, финансы)
- Битрикс24 (CRM данные)

## Масштаб
- 2000+ SKU сейчас, рост до 10-20 тысяч
- Еженедельная сборка: Ozon пн 00:00-14:00, WB пн 14:00-21:00
- Периоды: день, 3 дня, 7 дней, месяц, квартал, полгода, год, всё время

## Роли доступа (5 уровней)
1. **super_admin** — скрытый (Алексей). Видит ВСЁ. Не виден другим. Управляет кодом и системой.
2. **owner** — 2 собственника. Видят все данные/отчёты. Не видят код/техчасть. Не знают про super_admin.
3. **director** — управляющий директор. Основные метрики и данные по всей компании.
4. **head** — директора направлений. Видят ТОЛЬКО свои направления. Изолированы друг от друга.
5. **manager** — менеджеры. Строго то, за что отвечают (конкретные SKU/категории).

## Ключевые проблемы (решаем)
- Google Таблицы: лимит ячеек, таблицы множатся, теряется логика
- Долгая сборка по API (целый день по понедельникам)
- Сбои данных — пропустил одну сборку = всё криво
- Новые столбцы/строки в отчётах МП не отслеживаются
- Нет среза по отдельному товару / произвольному периоду
